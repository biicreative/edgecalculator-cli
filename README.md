# Edge Calculator CLI

[![Build Status](https://travis-ci.org/biicreative/edgecalculator-cli.svg?branch=master)](https://travis-ci.org/biicreative/edgecalculator-cli)

## Usage

```
$ edgecalculator-cli 

Options
    --url, -u   Set the calculator gateway

Examples

$ edgecalculator-cli --url $GATEWAY
```
